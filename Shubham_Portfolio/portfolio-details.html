This project aims to develop a robust and scalable online shopping platform for electronics using a microservices architecture. The application will be built using various technologies, including ASP.NET Core API, .NET Core, ASP.NET Core MVC, HTML, CSS, Bootstrap, and jQuery. The microservices architecture will enable the application to be highly modular and maintainable, allowing for easy scalability and future enhancements.
